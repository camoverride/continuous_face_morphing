# Continuous Face Morphing


## Setup

- `pip install -r requirements.txt`
- `pip install --upgrade setuptools`


## Run

For interactive mode:
- `python continuous_morph_from_camera.py`

For static mode:
- `python continuous_morph_from_directory.py`
